# Note Thumbnail Maker

note の記事編集画面で、記事タイトルからサムネイル画像を生成して、そのまま note のサムネイルとして挿入する Chrome 拡張です。

## ダウンロード

1. このリポジトリを ZIP でダウンロードするか、`git clone` で取得します
2. `note_thumbnail_maker` フォルダを手元に置きます

## インストール方法

1. Chrome で `chrome://extensions/` を開きます
2. 右上の `デベロッパーモード` を ON にします
3. `パッケージ化されていない拡張機能を読み込む` を押します
4. ダウンロードした `note_thumbnail_maker` フォルダを選びます

## 使い方

1. note の記事編集画面を開きます
2. `画像を追加` ボタンの近くに表示されるコントロールを使います
3. 必要に応じてフォント、サイズ、色を選びます
4. 背景画像を変えたい場合は `背景をアップロード` に PNG / JPG をクリックまたはドラッグ&ドロップします
5. note にそのまま入れる場合は `タイトルからサムネイルを生成` を押します
6. 画像だけ保存したい場合は `サムネをDL` を押します

## できること

- note の編集画面にサムネイル生成 UI を追加
- 記事タイトルをテンプレート画像の中央に描画して JPEG を生成
- 生成した画像を note のサムネイルとして挿入
- 生成した画像を JPEG でダウンロード
- インストール済みフォント一覧から描画フォントを選択
- 文字サイズと文字色を変更
- 背景画像を PNG / JPG から差し替え
- 背景画像は JPEG に変換して保存
- 背景画像の長辺が 1920px を超える場合は自動縮小

## 対象ページ

- `https://editor.note.com/notes/*/edit/`

対象外のページでは何もしません。

## 背景画像について

- 推奨サイズは `1920x1006` または `1280x670`
- アップロード可能な形式は `PNG` と `JPG / JPEG`
- 保存時に JPEG へ変換されます
- 長辺が `1920px` を超える画像は、縦横比を維持したまま縮小されます
- 背景画像を削除すると、拡張内のデフォルト背景に戻ります
- 背景画像は `chrome.storage.local` に保存されます

## タイトル改行について

- タイトルの改行は自動では行いません
- 改行したい位置に半角スペースを入れてください

## 権限

- `fontSettings`
  インストール済みフォント一覧を取得して、フォント選択 UI に表示するために使います
- `storage`
  背景画像の保存に使います

note_thumbnail_maker/

├── page-hook.js

## 注意

- note 側の UI 変更により動かなくなる可能性があります
- Chrome / Chromium 系ブラウザ向けです
- 成功通知は表示せず、失敗時のみエラーを表示します
